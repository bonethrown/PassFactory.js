PassFactory.js

PassFactory.js is a JavaScript library that can generate iOS 6 PassBook passes right within your web browser. Well, that's almost the truth. What it really does is generate a script (AppleScript) that you simply open and run on your Mac, and that generates a pass.

PassFactory.js is for making official passes from you. An iOS Developer License is required to generate the passes. If you want to make passes but don't care who they're signed by, check out PassSource.

Pass Designer

PassFactory.js comes with an online pass designer tool that you can use right here in your browser. Although this tool is hosted online, all of the work happens in your browser, with no information about your pass leaving your computer. If you have an iOS Developer License and a Mac, give it a try!

JavaScript Library

PassFactory.js is at its core a pure JavaScript libary. You can use it in your own sites and apps to create pass-generating scripts, or even extend it for more automated pass generation.
